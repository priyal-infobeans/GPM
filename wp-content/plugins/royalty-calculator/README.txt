== Description ==

Royalty Calculator is a WordPress plugin that allows you to generate reports by importing data from Excel files into MySQL.

== Installation ==

1. Upload the `royalty-calculator` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'GPM' menu in the WordPress admin to start exploring the plugin features.
4. Assuming the system has composer installed -> Inside the plugin directory `/wp-content/plugins/royalty-calculator/` locate 'PhpSpreadsheet' library folder. Install it by running composer update in command line.
